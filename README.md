# SMART_JUDGE
## 스마트 물류 시스템 구축을 위한 AI기반 식품 품질 판단 프로그램
- 건국대학교 스마트ICT융합공학과 2022년 2학기 졸업 프로젝트
- 개발 기간 : 2022.11.
- 개발 인원 : 3인 (AI, iOS, Server)
- 건국대학교 2022 SW경진대회(2022.11.14 ~ 16) **<장려상>** 수상   


### <AI 모델 설계 및 구현>
- python 3.8
- 구글 colab의 gpu 환경.
- AI framework : pytorch
- pre-trained CNN 모델 **VGG-16** 사용
- 최종 성능 Accuracy 90.70%
